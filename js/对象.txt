  对象 Object

什么是对象：生活中万物皆对象，对象是 一个具体的事物，如，一本书，一个人，一辆车，一个数据库，一个网页，一个服务器
在 JS 中：对象是一组无序的相关 属性 和 方法 的集合
属性：事物的 特征
方法：事物的 行为


创建 对象 的 3 种方式：
   利用 字面量 创建对象
   利用 new Object 创建对象
   利用 构造函数 创建对象


1、利用 字面量 创建对象
对象字面量：就是花括号{}里面包含了表达这个事物（对象）的属性和方法

如：
var obj = {};                        //创建了一个空对象

语法格式：
var 对象名 = {
   属性名1：属性值，
   属性名2：属性值，
   ...，
   方法名1：function(){
      //函数体
   }
}

注意：
多个属性或者方法之间用逗号隔开
方法冒号 后面跟的是一个域名函数 

1.2、调用对象属性：

1.2.1、对象名.属性名
console.log(对象名.属性名);               //输出：属性值
如：
console.log(obj.uname);

1.2.2、对象名['属性名']
console.log(对象名['属性名']);            //输出：属性值
如：
console.log(obj['uname']);

1.3、调用对象方法：

1.3.1、对象名.方法名（) ;
如：
obj.sayHi();


2、利用 new Object 创建对象

如：
var boj = new Object();               //创建了一个空对象           O 要大写
boj.uname = '张三';
boj.age = 18;
obj.sayHi = function(){
   //函数体
}

调用和前面一样


3、利用 构造函数 创建对象 （对象的实例化）
构造函数：就是把我们对象里面一些相同的属性和方法抽象出来封装到函数里面
注意：和函数不一样，里面不是普通代码，而是对象 

语法格式：
function 构造函数名（）{                 //创建构造函数            构造函数的名字首字母要大写
   this.属性 = 值；                            // this 表示当前的
   this.方法 = function（）{

   } 
}
new 构造函数名（）；                      //调用构造函数

如：
function Star(uname,age,sex) {
   this.name = uname;
   this.age = age;
   this.sex = sex;
}
var ldh = new Star('刘德华'，18，'男');

注意：
构造函数名的首字母要大写
我们构造函数不需要 return 就可以返回结果 （因为有 new 和 this 的关键字）
构造对象 泛指某一大累

关键字
new 构造函数可以在内存中创建一个空的对象
this 就会指向刚才创建的空对象


4、遍历对象

for...in 语句
for...in 语句 用于对 数组 或 对象 的属性进行循环操作（常用与对象）

基本格式：
for （变量 in 对象名）{

}

如：
for（var k in obj）{ 
   console.log(k);                           // k 变量 输出：属性名
   console.log(obj[k]);                   // obj[k] 输出：属性值
}
