jJavaScript 作用域：就是代码名字（变量）在某个范 内起作用和效果


1、全局作用域  和 局部作用域

1.1、全局作用域
定义：整个 script 标签，或者是 js 文件

如：
var num = 20;


1.2、局部作用域（函数作用域）
定义：在函数内部就是局部作用域在，只在函数内部起效果和作用

如：
function fu() {
   var num = 20;
}


2、全局变量 和 局部变量（全局变量 在浏览器关闭时才销毁 因此比较占内存，局部变量 在代码块运行结束后 就会消耗 更省空间）

2.1、全局变量
定义：在全局作用域下的变量

2.2、局部变量
定义：在局部作用域下的变量 或者 在函数内部的变量

注意：
1、如果在函数内部 没有声明直接赋值的变量也属于全局变量
2、函数的形参也可以看做是局部变量

如：
function fu() {
   num = 20;                  //没有声明直接赋值 属于 全局变量
}
console.log(num);          //输出：20

3.3、作用域链
定义：内部函数访问外部函数的变量，采取的是链式查找方式决定取值 （就近原则）

如：
var num = 10;

function fu () {
   var num = 20;
   function fun () {
      console.log(num);         // 输出：20
   }
}




注意：JS 作用域有 全局作用域 和 局部作用域 没有 块级作用域 {}
