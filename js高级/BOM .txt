
1、BOM ( Browser Object Model ) 即浏览器对象模型,它提供了独立于内容而与浏览器窗口进行交互的对象，其核心对象是 window



2、DOM 和 BOM 不同点

DOM：
文档对象模型
DOM就是把[文档」当做一个「对象」来看待
DOM的顶级对象是 document
DOM主要学习的是操作页面元素
DOM是 W3C 标准规范

BOM：
浏览器对象模型
把「浏览器」当做一个「对象」来看待
BOM的顶级对象是 window
BOM学习的是浏览器窗[口交互的一些对象
BOM是浏览器厂商在各自浏览器上定义的,兼容性较差



3、BOM 的构成
BOM 比 DOM 大，它包含DOM
window 包含了：document 、location 、navigation 、screen 、history

注意：window 对象是浏览器的顶级对象，它具有双重角色。



4、打印 window 对象：console.dir(window);

注意：window下的一个特殊属性：window.name
